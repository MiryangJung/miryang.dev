## Golang Wails Setup

고언어 Wails 기초 세팅

1. __go__ 설치
2. __node__ 설치
3. __gcc__ 설치
4. wails 설치 `go get -u github.com/wailsapp/wails/cmd/wails`
5. wails 설정
    - cmd 에서 `wails` 실행
    - 이름, 이메일 설정
6. `wails init` 실행 후 초기 설정
7. `wails build` 를 입력하면 build/test.exe 생성
8. 개발 모드 : `wails serve` 실행, frontend 폴더에서 `npm run serve` 실행

`function`  
- [Function](https://pursuit.purescript.org/builtins/docs/Prim#t:Function)
- `(->)` 는 문법 설탕으로 `Function` 대신 사용되는 게 권장

```
f :: Number -> Number
f :: Function Number Number
f :: (->) Number Number
```

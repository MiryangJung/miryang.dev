## 단순 타입

### Javascript에 대응하는 기본 유형 3가지

`Number`, `String`, `Boolean`

```purescript
> :type 1.0
Number

> :type "miryang"
String

> :type true
Boolean
```

### 내장 유형들

- `Int` : 소수점이 없음 = 정수
- `Char` : 작은 따옴표 사용

```purescript
> :type 1
Int

> :type 'A'
Char
```

- `Array` : 배열의 모든 요소는 같은 타입이어야 함

```purescript
> :type [1,10,20]
Array Int

> :type [1.0,2.0,3.0]
Array Number

> :type [1, 'a']
Error...
```

- `Records` : Js의 오브젝트

```purescript
> miryang = {name:"Miryang", favoriteNum: [1,5,10]}
> :type miryang
{ favoriteNum :: Array Int
, name :: String          
}   
```

## Quantified Types

- `forall` : parametric polymorphism(타입을 매개변수로 받아 새로운 타입을 되돌려주는 기능)
    - 함수의 인자가 type

```purescript
> :type flip
forall a b c. (a -> b -> c) -> b -> a -> c
```

## 유형 정의

street, city, state 필드가 String인 레코드 타입을 만들고, Entry라는 별칭을 가진다.  
레코드에는 다른 레코드가 포함될 수 있다.

```purescript
type Address =
  { street :: String
  , city   :: String
  , state  :: String
  }
type Entry =
  { firstName :: String
  , lastName  :: String
  , address   :: Address
  }
```

`List`를 만들 수 있다.  
List랑 Array는 다르다.

```purescript
type AddressBook = List Entry
```

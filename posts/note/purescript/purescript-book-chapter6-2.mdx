## [Show Me!](https://book.purescript.org/chapter6.html#show-me)

- Show 타입클래스
  - 문자열로 변환

```purescript
class Show a where
  show :: a -> String
```

boolean을 문자열로 변환

```purescript
instance showBoolean :: Show Boolean where
  show true = "true"
  show false = "false"
```

## [Common Type Classes](https://book.purescript.org/chapter6.html#common-type-classes)

- Eq 타입클래스
  - 같은지 비교
  - 유형이 같고, 값이 같아야 한다.

```purescript
class Eq a where
  eq :: a -> a -> Boolean
```

- 모노이드
  - Again, strings and arrays are simple examples of monoids.
  - empty 값을 이용해 자기 자신을 만들 수 있다?

```plain
"abc" append "" = "abc"
[a,b,c] append [] = [a,b,c]
```

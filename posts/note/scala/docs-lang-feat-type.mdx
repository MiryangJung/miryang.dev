1. sbt 설치 `brew install sbt`
2. [Scala (Metals)](https://marketplace.visualstudio.com/items?itemName=scalameta.metals) 확장프로그램 설치
3. Scala 프로젝트를 열면 뜨는 `Import build` 를 클릭해서 설치
